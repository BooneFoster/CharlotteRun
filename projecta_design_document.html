Project A: Charlotte Run Design Page

  1. Project Overview
    • My application is a running webpage that serves as a community project for my Dad.
    • The inteneded users of this site are myself, my dad, and members of the Charlotte running community.
    • The website will include five pages. These pages will include a way to add runs to the joint charlotte run list, an announcement section for running events in the community, a gallery of photos of charlotte runners, and finally a weekly charity that the community can learn about.
